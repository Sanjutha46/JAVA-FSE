# Exercise 1 - Mocking and Stubbing with Mockito

Objective:
To learn how to create mock objects and stub methods using Mockito.

Concepts Used:
- mock()
- when()
- thenReturn()
- assertEquals()

Scenario:
A service depends on an external API. The API is mocked so that unit testing can be performed without calling the real API.

Expected Result:
The mocked API returns "Mock Data" and the test passes successfully.
