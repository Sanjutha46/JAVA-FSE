# E-Commerce Platform Search Function

## Objective
Implement Linear Search and Binary Search to search products in an e-commerce platform.

## Product Attributes
- productId
- productName
- category

## Algorithms Used
1. Linear Search
2. Binary Search

## Time Complexity

Linear Search:
- Best: O(1)
- Average: O(n)
- Worst: O(n)

Binary Search:
- Best: O(1)
- Average: O(log n)
- Worst: O(log n)

## Conclusion

Binary Search is more efficient for large product datasets because it reduces the search space by half in every iteration.
