# Financial Forecasting

## Objective

Predict future financial values using recursion.

## Formula

Future Value = Current Value × (1 + Growth Rate)

## Example

Current Value = 10000

Growth Rate = 10%

Years = 5

Future Value = 16105.10

## Recursion

The recursive function repeatedly applies the growth rate until the number of years becomes zero.

## Time Complexity

O(n)

## Space Complexity

O(n)

## Optimization

1. Iterative approach
2. Mathematical formula using exponentiation
